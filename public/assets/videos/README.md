# Drop high-quality videos here

This is the home for your good video files (launch videos, reels, hero clips).
Anything you put in this folder is live on the site at:

```
https://fade.bet/assets/videos/YOUR-FILE-NAME.mp4
```

## How to add a video (from the GitHub website — no coding)

1. Open this folder on GitHub: **public → assets → videos**
2. Click **Add file → Upload files**
3. **Drag your video in** (or click "choose your files")
4. Scroll down, click **Commit changes**
5. Tell me the file name and where you want it used, and I'll wire it up.

## Naming

Use simple, lowercase names with dashes — no spaces, no capitals:

- ✅ `launch-video.mp4`
- ✅ `nate-hero.mp4`
- ❌ `Launch Video FINAL (2).mov`

## Format & size — read this so videos actually play

- **Use `.mp4` (H.264 video + AAC audio).** This is the format every phone and
  browser plays. Most exports default to it. Avoid `.mov` when you can.
- **Keep files under ~50 MB** if possible. Big files load slowly for viewers and
  GitHub's website upload caps at **25 MB per file**.
- **Over 25 MB?** You can't drag it into the website. Send it to me (or drop it
  in Google Drive) and I'll add it for you — I can also compress a huge file
  down without losing visible quality.
- **Over 100 MB?** GitHub blocks these without special setup. Don't upload it —
  tell me and I'll set up large-file storage.

## Want a video swapped into a page?

Just tell me the file name + which page (e.g. "use `launch-video.mp4` on the
media gate"), and I'll point the page at it.
